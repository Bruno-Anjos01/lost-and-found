<img src="home.png" width="300">

## 📖 Descrição
Este projeto é uma aplicação web desenvolvida em Python com Flask
que permite cadastrar, visualizar e gerenciar itens.

## ✨ Funcionalidades
- Cadastro de iten
- Edição de itens
- Remoção de itens
- Feed de itens
- Sistema de login de usuário
- Sistema cadastro de usuário

## 🛠️ Tecnologias
- Python 3.13
- Flask
- HTML5
- CSS3
- Bootstrap
- SQLite
